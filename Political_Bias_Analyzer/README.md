# Political Bias Analyzer

A web-based tool designed to analyze political bias in online articles and content. Simply paste a URL to analyze the political bias of the content.

## Features

- **URL Input**: Paste any URL to analyze its content
- **Simple Interface**: Clean, modern user interface with intuitive controls
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Keyboard Support**: Press Enter to trigger analysis

## Current Status

This is an early-stage project. The UI is complete, but the analysis functionality is still under development.

## How to Use

1. Open `index.html` in your web browser
2. Paste a URL into the input box
3. Click the "Analyze" button (or press Enter)
4. Use the "Reset" button to clear the input and start over
